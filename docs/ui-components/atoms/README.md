# Atoms

Single-purpose UI primitives. An atom does one thing and has no internal composition of meaningful sub-components.

Examples: button, input, label, icon, badge, spinner, divider, tooltip.

Real atoms live in this folder, one file per component (`kebab-case.md`). See [`../_templates/atom-example-button.md`](../_templates/atom-example-button.md) for the shape of a filled-in component spec — reference only, not project content.
