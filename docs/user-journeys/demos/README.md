# Journey Demos

Static HTML mockups for each journey step, linked from the journey Markdown files. These are throwaway scaffolding to support on-site role-play sessions — not production code.

Naming: `NN-{step-slug}.html`, matching the journey number, e.g. `01-lead-capture.html`.

Once the Blazor Journey Index prototype is scaffolded (`/prototypes/`), these HTML demos can be migrated or replaced.
